About

This tool allows you to visualize the structure of PyTorch models saved in the SafeTensor format. Features include:

  • Interactive graph visualization of model layers
  • Hover over nodes to see tensor details
  • Click on nodes to view in-depth tensor statistics
  • Dark theme UI for comfortable viewing
